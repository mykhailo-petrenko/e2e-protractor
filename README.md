## Prerequisites
java 1.8.x
node 6.x
protractor
mocha
chai
chai-as-promised

## Installation guide
npm i -g protractor

set NODE_TLS_REJECT_UNAUTHORIZED=0
webdriver-manager update

## Start testing
webdriver-manager start
